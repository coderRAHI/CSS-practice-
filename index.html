Methods of Adding CSS in any HTML file.

  1. Inline CSS:- Inline Css is used by the style attribute in a required HTML tag.
  2. Internal CSS:- Internal CSS is used by using style tag which lies inside the head tag.
  3. External CSS:-External CSS is used by creating a external file and link it with the HTML file by using "link:css" tag in html.
